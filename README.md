# Metadata Publication and Access Working Group

This repository organizes reviews – conducted as described in [Development and Review Processes for DDI Standards and Official Technical Documents](https://doi.org/10.5281/zenodo.7765072) – of the outputs from the [Metadata Publication and Access Working Group](https://ddi-alliance.atlassian.net/wiki/x/AQDk4) prior to publication.

The first document to be reviewed will most likely be the [Application for a Media Type](applications_media_type/ddi-codebook.md) for DDI-Codebook XML. The timeline and details on who will be invited to comment, when, and how will be provided here.
